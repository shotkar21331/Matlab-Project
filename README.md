Real-Time Surrounding Audio Event Detection
Abstract

This project presents a real-time surrounding audio event detection system designed for indoor environments using deep learning techniques. Sound plays a vital role in human daily life, and humans can naturally recognize and differentiate various sounds with ease. However, enabling computers and intelligent systems to understand and classify environmental sounds remains a challenging task.

The proposed system focuses on recognizing real-time surrounding audio events through machine learning and audio signal processing. Since audio is continuous in nature, unlike text or images, extracting meaningful audio features and performing accurate recognition in real time is a complex problem. To address this, the system captures audio input, extracts relevant audio features, and performs synchronized acquisition and recognition for accurate predictions.

The model was trained using a standard audio dataset with a Feed-Forward Neural Network (FFNN) architecture. A Softmax activation function was used for multi-class sound classification. The system is capable of identifying different environmental sounds in real time and producing predictions efficiently.

The primary objective of this project is to record and classify surrounding sounds using deep learning techniques and evaluate the system’s real-time performance. Experimental results achieved an overall accuracy of 80%, demonstrating the effectiveness of the proposed approach for real-time audio event detection applications.
